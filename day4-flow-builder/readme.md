FLOW BUILDER:
Flow Builder is a no-code automation tool
Used to automate business processes
Works using logic + conditions
✔ Example:
Flow Builder is a Salesforce tool that allows users to automate business workflows without coding. It helps in reducing manual work and improving efficiency.

Types of Flows :
Screen Flow:
Requires user input
Has UI (screens)
👉 Example:
Registration form
Record-Triggered Flow:
Runs automatically when record changes
👉 Example:
When Lead is created → send email

Automation Ideas:
Auto email after student registration
Notify faculty when course is full
Generate student ID automatically
Send fee reminder before deadline
Update available seats automatically

Flow Diagram:
Start → Condition → Action → End
example:
Trigger: Student registers
↓
Check: Seats available?
↓
Yes → Confirm admission
No → Send waiting list email

Manual vs Automation Process :
Example: Fee Payment Reminder
Manual:
Staff checks records
Sends emails manually
Problems:
Time-consuming
Errors possible
Delays
Automated:
System sends reminders automatically

Reflection  :
Why automate?
Saves time
Reduces errors
Improves efficiency
When NOT to automate?
Complex human decisions
Low-frequency tasks
